🎮 Game Dev Roadmap

Parcours vers la création d’un studio de jeux indépendant.

📌 Phase 1 — Fondations

  • HTML / CSS / JavaScript
  • Python (logique + scripts)
  • Algorithmique de base
  • Comprendre les moteurs de jeu (Unity / Unreal)

🧠 Phase 2 — Prototypage

  • Créer des mini-jeux 2D
  • Prototype gameplay (infiltration, combat, choix)
  • Tester des systèmes de décisions dynamiques
  • Apprendre les bases du level design

⚙️ Phase 3 — Projets sérieux

  • Développer un jeu complet jouable
  • IA basique pour PNJ
  • Systèmes de progression et sauvegarde
  • Optimisation et performance

🏢 Phase 4 — Studio Indie

  • Créer un studio indépendant
  • Publier sur Steam / itch.io
  • Construire une communauté
  • Monétisation et projets commerciaux

🚀 Vision finale

Créer des jeux narratifs immersifs où chaque décision a un impact réel sur le monde, les factions et les personnages.
