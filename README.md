# Portfolio Adam

Portfolio personnel avec une direction plus creative et plus orientee game dev.

## Contenu

- Une page d'accueil moderne, legere et visuellement plus premium
- Une section identite pour presenter le profil et la vision
- Une section competences plus claire
- Une separation entre projets tech et concepts de jeux
- Une section Game Studio avec trois pitches courts et accrocheurs
- Une page Roadmap pour montrer l'objectif de creation d'un studio indie

## Style

Le site adopte une direction elegante inspiree d'une finition Apple :

- verre floute
- cartes lumineuses
- animations douces
- navigation propre
- theme sombre simple
